Helen Chen’s Data550 Final Project Outline
================
Hua Xuan (Helen) Chen

There are three main folders in the project directory: output, data, and
code. The output folder stores the outputs generated by the R scripts in
the code folder; they are in rds and png formats. The data folder
contains the shoe data in csv format. The project folder also contains a
Report.Rmd, a .gitignore file, a Makefile, and the README.

You may enter “make” in the terminal, which will generate the final
report in html format. The current contents of the report are: EDA table
and 2 boxplots, coefficients from LASSO for feature selection in linear
regression to find the most important predictors of revenue, and results
from a linear mixed effects model.

## Code description:

`code/make_table.R`

- This code creates the table that shows the total shoe units sold and
  total revenue (in USD) by country, in increasing order of total
  revenue.

- Saves the table as a `.rds` object in `output/` folder

`code/render_report.R`

- Renders `Report.Rmd`

`code/make_boxplots.R`

- This code creates two boxplots:

- First boxplot shows the distribution of revenue for each of the shoe
  brands in the dataset, in increasing order of median revenue

- Second boxplot shows the distribution of revenue for each of the shoe
  categories, in increasing order of median revenue

- Saves the boxplots as `.png` files in `output/` folder

`code/model.R`

- This is still a work in progress, but it currently uses LASSO for
  feature selection in linear regression to find the most important
  predictors of revenue, as there are numerous predictors in this
  dataset

- Saves the matrix of coefficients from LASSO output as a `.rds` object
  in `output/` folder

- Also, I ran a linear mixed effects model with a random intercept for
  units sold and the following predictors: brand, category, gender of
  consumer, size, color, discount percentage, sales channel, country,
  payment method, customer income level, customer rating, and year

- The output from the model above is saved as a `.rds` object in
  `output/` folder

`Report.Rmd`

- Displays a summary table, 2 boxplots, coefficients from LASSO, and
  summary of linear mixed effects model with a random intercept for
  units sold
